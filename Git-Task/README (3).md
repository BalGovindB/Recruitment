# Git Task — Git Mastery

## Task Description

Complete the Beginner and Advanced tracks on [gitmastery.me](https://gitmastery.me/), an interactive game that teaches Git commands and concepts level by level instead of just reading documentation about them.

## How I Implemented It

Worked through each module in order — Introduction to Git, File Operations, Working with Branches, Remote Repositories — clearing every level in a module before moving to the next. Git Mastery checks your actual commands against the expected repo state rather than just multiple choice, so it forced me to actually type out `git add`, `git commit`, `git branch`, `git merge`, etc. instead of recognizing them.

## What I Learned

- Working with Branches went from something I could describe but not really do, to something I could actually execute confidently — creating, switching, and merging branches, and dealing with the occasional merge conflict.
- Remote Repositories made the difference between `git fetch` and `git pull` finally click — I used to just always reach for `pull` without knowing it's fetch + merge combined.
- Seeing the same operation fail when I did it in the wrong order (e.g. trying to push before pulling) was a better teacher than any explanation of why you need to sync first.
